== Description ==

Easiest way to bulid up a membership site.

You can place Sign up and Sign in buttons anywhere on your site with shortcode.

Sign up and Sign in forms are displayed as a modal window, no special page required.

This plugin enable you to display different contents for guests and members on any posts and pages you would like to.

== Installation ==

1. Download the plugin file
2. Unzip the file and upload into /wp-content/plugins/
